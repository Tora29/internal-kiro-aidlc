# どんMai — サボりを肯定する自己正当化Webアプリ

遅刻した朝、締切を破った夜、人は言い訳を考えるストレスと罪悪感を同時に抱える。「どんMai」は、そのしんどさをAIに丸投げできるWebアプリだ。たぬきのマスコット「どんMai」に状況を打ち込むだけで、もっともらしい言い訳が3秒で返ってくる。

使えば使うほど自分で考えなくなる。それがこのサービスの狙いであり、「人をダメにするサービス」というテーマへの直接的な回答でもある。

---

## 主な機能

### 言い訳生成
Amazon Bedrock（Claude Haiku）が状況に応じた言い訳を瞬時に生成する。言い訳を考える手間ごとAIに肩代わりさせる。

- **Free プラン**: 標準品質（Claude Haiku）
- **Pro プラン**: より破綻しにくい高精度な言い訳（品質不足の場合のみ Claude Sonnet にフォールバック）

### 懺悔室
言い訳を使ってその場をしのいだあと、サボった後ろめたさをどんMaiに打ち明けると「それは仕方なかった」「むしろよくやった」と全力で受け止めてくれる。チャット形式UIで、過去の言い訳履歴から選択して懺悔することも可能。

### サボり肯定ポップアップ
30秒間無操作でどんMaiが「休んでて偉いですね！」等のメッセージを表示する。`mousemove` / `keydown` イベントで検知（タブ非アクティブによる発火はなし）。

### オンボーディング（5ステップ）

| ステップ | 内容 | 認証要否 |
|---------|------|---------|
| Step 1 | ようこそ画面 / サービス説明 | 不要 |
| Step 2 | デモ体験（言い訳を1回無料生成・認証なし） | 不要 |
| Step 3 | アカウント登録 / ログイン + Free・Pro プラン選択 | ここで認証 |
| Step 4 | シーン選択（ビジネス・友達・カレカノ・学校・家族） | 必要 |
| Step 5 | 分析結果に合ったどんMaiが登場 | 必要 |

シーン選択に応じてたぬきのキャラクターがパーソナライズされ、自分専用のたぬきとして愛着が生まれる設計。

## マスコットキャラクター「どんMai」

| バリエーション | 表示条件 |
|-------------|---------|
| デフォルトたぬき | 未選択時・汎用 |
| スーツたぬき | ビジネスシーン選択時 |
| 私服たぬき | 友達シーン選択時 |
| おめかしたぬき | カレカノシーン含む選択時 |
| 撃たれたぬき | 攻撃入力時 |

「お前を消す方法」などの有名ミームを入力した場合、「たぬきはもうおしまいです」系のミーム返答あり。

## プラン設計

| | Free | Pro |
|--|------|-----|
| 料金 | 無料 | 月額 1,000〜2,000円 |
| 利用制限 | 1日 10,000〜15,000 トークン | 無制限 |
| 言い訳品質 | 標準（Claude Haiku） | 高精度モード解放 |
| たぬきキャラ | デフォルトのみ | 全バリエーション |
| 課金システム | — | AWS Marketplace |

トークン上限到達の瞬間に Pro への誘導モーダルを表示し、最も「もっと使いたい」と感じているタイミングでアップセルにつなげる。

## 技術スタック

### フロントエンド
- SvelteKit（SSG / adapter-static）
- Tailwind CSS
- Svelte アニメーション

### バックエンド
- AWS Lambda（Python + Boto3）
- Amazon API Gateway（Cognito オーソライザー）
- Amazon Bedrock Converse API（Claude Haiku / Sonnet）
- Amazon DynamoDB（ユーザー設定・トークン使用量・言い訳履歴）
- Amazon Cognito（メール+パスワード / Google ソーシャルログイン）
- AWS CDK（IaC）

### ホスティング・配信
- Amazon CloudFront + S3（SvelteKit 静的ファイル）
- Amazon CloudWatch（ログ・監視）
- AWS CodePipeline + CodeBuild（CI/CD、main ブランチ push でトリガー）

### アーキテクチャ概要

```
フロントエンド（SvelteKit / CloudFront + S3）
        ↓
API ゲートウェイ（認証検証 / ルーティング）
        ↓
バックエンド（AWS Lambda マイクロサービス）
  ├─ 言い訳生成サービス → Amazon Bedrock
  ├─ 懺悔室サービス    → Amazon Bedrock
  ├─ 認証サービス      → Amazon Cognito
  ├─ トークン管理      → DynamoDB
  ├─ プラン管理        → AWS Marketplace
  └─ キャラクター管理  → DynamoDB
```

## 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンス | 言い訳生成は 3秒以内（Bedrock 呼び出し含む） |
| 可用性 | Lambda + API GW のマネージド構成で 99.9% 以上 |
| セキュリティ | Cognito JWT 認証・IAM 最小権限・基本的なプロンプトインジェクション対策 |
| スケーラビリティ | Lambda オートスケール・DynamoDB オンデマンド |
| プライバシー | 入力テキストは CloudWatch ログに残さない |
