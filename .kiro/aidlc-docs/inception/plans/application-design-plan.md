# Application Design Plan - どんMai

## 概要
このプランは、「どんMai」プロジェクトのアプリケーション設計を定義します。主要なコンポーネント、責務、インターフェース、サービス層、依存関係を明確にします。

---

## 設計スコープ

### 主要なビジネス機能
1. **オンボーディング**: 5 ステップのユーザーオンボーディング
2. **言い訳生成**: Free/Pro 品質切り替え、トークン管理
3. **懺悔室**: 肯定メッセージ生成
4. **サボり肯定ポップアップ**: 無操作検知、メッセージ表示
5. **認証・プラン管理**: Cognito 認証、プラン切り替え
6. **キャラクター管理**: キャラクター分岐、表示ロジック
7. **攻撃入力検知**: プロンプトインジェクション対策

### 技術スタック
- **フロントエンド**: SvelteKit + Tailwind CSS
- **バックエンド**: AWS Lambda + Python
- **AI**: Amazon Bedrock (Claude Haiku/Sonnet)
- **認証**: Amazon Cognito
- **データベース**: Amazon DynamoDB
- **ホスティング**: CloudFront + S3
